## Project Description

This project is a simple Login and Registration system implemented in C++. It allows users to register with a username and password, securely store their credentials, and log in with proper validation.

### Features

- User registration with username and password.
- Input validation and username duplication check.
- Securely stores user credentials in a file.
- Login functionality with verification of stored credentials.
- Displays appropriate success or error messages.

---

## Technologies Used

- C++ (Standard Template Library, File Handling)
- Command-Line Interface (CLI)

## How to Run

1. **Compile the code** using any C++ compiler:  
   ```bash
   g++ -o login_system main.cpp
   ```

2. **Run the executable**:  
   ```bash
   ./login_system
   ```

3. Follow on-screen instructions to Register or Login.
